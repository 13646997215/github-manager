# Code of Conduct

## Our Pledge
We want ROS2-Agent to be a serious, welcoming, engineering-focused open-source project.

## Expected behavior
- Be respectful and technically honest.
- Prefer evidence over ego.
- Critique ideas and code, not people.
- Help newer ROS2 users learn instead of shaming them.
- Keep discussions constructive and relevant.

## Unacceptable behavior
- harassment, intimidation, or personal attacks
- discriminatory language or exclusionary behavior
- malicious sabotage of contributions or reviews
- knowingly misleading technical claims
- publishing private information without permission

## Enforcement
Maintainers may reject, remove, or moderate contributions and discussions that violate this policy.

## Reporting
For conduct concerns, use the private reporting path described in SECURITY.md.
