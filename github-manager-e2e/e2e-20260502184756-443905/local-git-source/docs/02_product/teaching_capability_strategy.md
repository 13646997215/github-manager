# Teaching Capability Strategy

ROS2-Agent should not only diagnose problems, but also help users become better ROS2 engineers.

## Teaching differentiators
- explain the why behind each check
- separate symptom, observation, root-cause hypothesis, and next action
- progressively build user independence
- treat common ROS2 historical pain points as teachable cases

## Teaching asset types
- teaching-focused skills
- guided transcripts
- benchmark cases that explain why an answer is strong or weak
- learning-track docs for beginners and lab newcomers
