# ROS2-Agent 产品定位

## 面向谁

1. ROS2 初学者
2. 机器人仿真开发者
3. 实验室 / 竞赛团队成员
4. 小型机器人研发团队

## 解决什么问题

- 环境脆弱
- 老手经验难复用
- 排障链条长
- 工作流重复且容易漏验证
- 缺少真正会执行的机器人开发 Agent

## 核心卖点

- 基于 Hermes 的长期记忆与自动化底座
- 针对 Ubuntu 22.04 + ROS2 Humble 深度设计
- 强调技能库、结构化工具、测试基准、可复现实例
- 目标是开发平台，而不是普通问答机器人

## 产品主张

ROS2-Agent 应当成为：

“一个真正懂 ROS2 开发现场、会教学、会排障、会自动化、会不断沉淀经验的专家平台。”
