# ROS2 Skills Taxonomy

## Layer 1 - Foundation Skills
- ros2-environment-bootstrap
- ros2-workspace-diagnose
- ros2-dependency-repair

## Layer 2 - Build and Launch Skills
- ros2-colcon-build-troubleshoot
- ros2-launch-debug
- ros2-runtime-log-analysis

## Layer 3 - Simulation Skills
- ros2-gazebo-debug
- ros2-rviz-debug
- ros2-tf-qos-network-debug
- ros2-controller-manager-debug

## Layer 4 - Quality and Automation Skills
- ros2-regression-report
- ros2-benchmark-runner
- ros2-demo-walkthrough

## Skill Quality Standard
Each skill must include:
- use cases
- prerequisites
- numbered workflow
- failure branches
- verification steps
- risk notes
- escalation conditions
