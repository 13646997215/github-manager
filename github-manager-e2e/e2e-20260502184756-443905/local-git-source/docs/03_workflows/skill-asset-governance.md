# Skill Asset Governance

## 目标
让 repo skills 成为可检查、可导出、可验证的资产集合。

## 当前工具
- scripts/skills/inspect_repo_skills.py
- scripts/skills/export_skill_manifest.py
- scripts/skills/validate_skill_metadata.py

## 当前输出
- profile/ros2-agent/skill_manifest.json

## 说明
当前仍是 repo-scoped manifest/validation，不代表已自动同步到 Hermes 全局 skill storage。
