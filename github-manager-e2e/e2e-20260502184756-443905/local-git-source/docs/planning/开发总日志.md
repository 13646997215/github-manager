## 0. 项目总目标

- Ubuntu 22.04 + ROS2 Humble 仿真开发
- 教学引导与工程实践
- 技能体系与结构化工具体系
- Profile / Gateway / Cron / MCP / Benchmark / Examples / Tests
- 面向 GitHub 社区的高竞争力开源项目形态

1. 严格使用项目工作区，不乱放文件。

4. 优先保证高质量、可解释、可验证、可维护。
