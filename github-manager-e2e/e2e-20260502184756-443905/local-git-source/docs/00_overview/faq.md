# FAQ

## Is ROS2-Agent just another chatbot?
No. It is a repository-backed ROS2 expert platform with skills, structured tools, fixtures, benchmarks, tests, and validation pipelines.

## Why is it repository-backed?
Because transparency, inspectability, and reproducibility are more valuable than hiding logic in an opaque agent bundle.

## Why only Ubuntu 22.04 + ROS2 Humble first?
Because narrow verified support is better than broad fragile claims.

## Does it already solve runtime problems or only build errors?
It already includes runtime graph, controller, and TF-oriented structured reasoning prototypes and benchmark fixtures, and continues to expand in that direction.

## Is this meant for real robots or simulation?
The current default posture is simulation-safe and diagnosis-first. Real-hardware paths should remain explicit and high-caution.
