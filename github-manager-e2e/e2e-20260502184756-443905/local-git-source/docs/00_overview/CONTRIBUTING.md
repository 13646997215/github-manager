# Contributing to ROS2-Agent

## Current contribution focus

At the current stage, the repository especially welcomes contributions in:
- ROS2 domain skills
- structured tool contracts
- benchmark cases
- broken-case fixtures
- docs and reproducible workflows

## Contribution principles

1. Keep files in the correct directory.
2. Prefer reproducible examples.
3. Add validation or tests where possible.
4. Update documentation when adding reusable assets.
5. Preserve the simulation-safe default posture.
