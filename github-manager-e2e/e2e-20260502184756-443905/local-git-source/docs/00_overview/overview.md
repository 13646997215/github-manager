# ROS2-Agent 概览

ROS2-Agent 是一个基于 Hermes Agent 框架构建的 ROS2 垂直专家平台，面向 Ubuntu 22.04 + ROS2 Humble 的仿真开发、教学引导、测试回归与工程自动化。

## 平台目标

构建一个真正具备以下能力的机器人开发平台：

- 会教学：解释 ROS2 基础与工程实践
- 会执行：辅助环境检查、workspace 诊断、构建与排障
- 会记忆：长期保存高价值经验和项目约定
- 会自动化：支持定时检查、回归、报告生成
- 会扩展：支持 skills、tools、gateway、cron、mcp、多 agent 协作

## 平台边界

短期聚焦：
- Ubuntu 22.04
- ROS2 Humble
- 仿真开发、安全调试、测试验证

中长期扩展：
- 实验室协作
- CI / 回归平台
- Gateway / 远程消息协作
- MCP 外部系统接入
- 多 Agent 深度协作

## 关键设计思想

1. Hermes 做底座，ROS2-Agent 做领域专家层。
2. Skill-first 优于一开始堆大量复杂工具。
3. 高频流程逐步升级为结构化工具输出。
4. 每项能力都要可验证、可复现、可文档化。
5. 最终目标是完整平台，而不是仅仅完成 MVP。
